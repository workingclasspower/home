* 资本家知道一个人的年轻时光是生产率最高的，所以他们鼓吹年轻人要996，要奋斗，要坚持，但从来不谈给你的回报。而且一旦你年龄增大，产出下降，没有剥削价值，他们就会用早已设计好的各种制度攻击你，像扔垃圾一样将你淘汰。
